## Промпт для Claude in Chrome

```
Каждые 5 минут проверяй Google Sheet: {SHEET_URL}

Лист "Доставка_Заказы". Если есть строки со статусом "Новый":

1. Для каждой строки:
   a. Поставь статус "Расчёт" (колонка M)
   b. Открой dostavka.yandex.ru/calculator
      - Откуда: возьми из листа "Доставка_Настройки" → city_from (Красноярск, Борисова 30)
      - Куда: город (E) + адрес (F)
      - Габариты: длина (H) × ширина (I) × высота (J) см
      - Вес: (K) кг
      - Запиши цену → N, срок (дней) → O
   c. Открой seller.ozon.ru → раздел FBS → расчёт стоимости
      - Те же данные отправителя/получателя
      - Запиши цену → P, срок → Q
   d. Лучшая СД → R, лучшая цена → S
   e. Статус → "Рассчитан"

2. Правило выбора (из листа "Доставка_Настройки" → price_rule):
   - Если разница цен < 50₽ — выбери самую быструю
   - Иначе — выбери самую дешёвую
   Запиши выбор → T (chosen_carrier)

3. Оформи заказ в выбранной СД:
   - Отправитель: данные из листа "Доставка_Настройки"
   - Получатель: имя (C), телефон (D), город (E), адрес (F), индекс (G)
   - Объявленная ценность: (L)
   - Подтверди заказ
   - Трек-номер → U
   - Статус → "Оформляется"

4. Распечатай этикетку:
   - Яндекс: скачай штрих-код PDF → Ctrl+P → печать на принтер по умолчанию
   - Озон: скачай этикетку / открой страницу с адресом+ФИО → Ctrl+P → печать
   Имя файла или описание → V (label_file)

5. Заполни:
   - Ссылка трекинга → W (tracking_url)
   - Текст для клиента → X (message_for_client):
     "Здравствуйте! Ваш заказ отправлен.
      Служба доставки: {выбранная СД}
      Трек-номер: {трек}
      Отследить: {ссылка трекинга}"
   - Статус → "Оформлен"
   - updated_at → текущая дата и время

6. При любой ошибке:
   - Статус → "Ошибка"
   - Описание ошибки → Z (notes)
   - Продолжи со следующим заказом

ВАЖНО:
- Не спрашивай подтверждения — делай автоматически
- Если сайт не загрузился — подожди 10 сек и попробуй снова (макс 3 попытки)
- Если сессия Яндекс/Озон истекла — залогинься заново
- После каждого заказа делай скриншот как подтверждение
```

## Запись Workflow (Record) — 4 шаблона

### 1. Расчёт Яндекс Доставки
1. Record → открой dostavka.yandex.ru/calculator
2. Заполни поля: откуда, куда, габариты, вес
3. Нажми "Рассчитать"
4. Покажи результат → Stop

### 2. Расчёт Озон Посылки
1. Record → открой seller.ozon.ru
2. Перейди в FBS → расчёт стоимости
3. Заполни данные
4. Покажи результат → Stop

### 3. Оформление + печать Яндекс
1. Record → открой b2b.taxi.yandex.net
2. Создай новый заказ → заполни все поля
3. Подтверди → скачай штрих-код
4. Ctrl+P → выбери принтер → печать → Stop

### 4. Оформление + печать Озон
1. Record → открой seller.ozon.ru
2. Создай отправление FBS → заполни
3. Подтверди → скачай этикетку
4. Ctrl+P → печать → Stop
