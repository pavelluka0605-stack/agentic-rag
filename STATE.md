## Известные баги

1. **yandex-metrika.yml** — shell-переменные (`$SUMMARY`, `$SOURCES` и т.д.) не попадают в `python3 << 'PYEOF'` через `os.environ`, потому что они не экспортируются (`export`). Нужно либо добавить `export` перед каждой переменной, либо передавать данные через временные файлы.

2. **Google APIs** — заблокированы из среды Claude Code (403). Все операции с Sheets — через GitHub Actions.
