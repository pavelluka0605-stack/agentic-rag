# Визуальное направление — от MEBELIT.site к «Кухни Рема»

**Дата:** 2026-03-15
**Статус:** Утверждение направления
**Зависимости:** `DISCOVERY-AUDIT.md`, `brand-guide.md`, `design-tokens.css`

---

## 1. Executive Summary

**Что берём от MEBELIT.site:**
- Структуру блоков: Hero с фото кухни + overlay, каталог карточками, квиз-калькулятор, секция команды, блоки преимуществ
- Принцип прозрачных цен на каждой карточке
- Прямоугольные карточки без скруглений (кухни = чёткие линии, не soft UI)
- Концепцию тёмного Hero как драматичного первого экрана
- Montserrat для заголовков (преемственность типографики)

**Что эволюционируем:**
- Цветовую схему: золотой (#e8c547) уходит, приходит терракот (#C2613A) -- тепло вместо «гламура»
- Тёмный фон (#1a1a2e) остаётся только в Hero и акцентных секциях, контентные блоки -- светлые
- Типографическую иерархию: добавляем чёткие уровни h1-h4 с контрастными размерами
- Сетку: хаотичные отступы заменяем на 8px-систему с design tokens
- Фото-контент: 3D-рендеры заменяем реальными фото из VK (205 фото, 42 модели)
- Отзывы: фейковые (30 шт., все 5 звёзд) заменяем реальными из Flamp/2GIS
- CTA: с ~10 точек сокращаем до 4 стратегических

**Что точно убираем:**
- Золотой (#e8c547) -- ассоциируется с дешёвым «гламуром», не с теплом кухни
- Англоязычный мусор («A TRUE NORTHERN PLAYA»)
- Дублирование Hero (десктоп + мобильная версия как отдельные секции)
- 3D-рендеры как единственный визуальный контент
- Перебор форм и попапов
- Фейковые отзывы

## 3. Три направления эволюции

### Направление A: «Чистый тёмный» (Clean Dark)

**Концепция:** Сохраняем тёмную базу MEBELIT как доминирующую, но очищаем и утепляем.

**Палитра:**
- Фон: #1C1C1C (нейтральный тёмный) вместо #1a1a2e (холодный с синевой)
- Альтернативный фон: #2D2418 (тёплый тёмно-коричневый для чередования секций)
- Акцент: #C2613A (терракот) вместо #e8c547 (золотой)
- Текст: #F5EDE3 (тёплый белый) вместо #FFFFFF (чистый белый)
- Карточки: #2A2A2A (тёмно-серый) с border #3A3A3A

**Визуальная логика:**
- Все секции на тёмном фоне
- Фото кухонь контрастно выделяются (светлые кухни на тёмном -- эффектно)
- CTA-кнопки терракотовые -- яркое пятно на тёмном
- Текст светлый, иерархия через размер и weight

**Что сохраняется от MEBELIT:** тёмный характер, серьёзность, контрастность, ощущение «не дешёвый сайт».

**Что улучшается:** убираем холодный синеватый подтон (#1a1a2e), добавляем тепло через коричневые оттенки и терракотовые акценты.

**Плюсы:**
- Максимальная визуальная преемственность с MEBELIT -- клиенты узнают «свой» сайт
- Кухни (особенно светлые фасады) выглядят выигрышно на тёмном фоне
- Тёмные сайты визуально воспринимаются как «дороже» -- для сегмента 175K-429K это плюс
- Выделяется среди конкурентов (все на светлом фоне)

**Минусы:**
- Тёмные сайты хуже для длительного чтения -- блог, описания моделей, FAQ будут утомлять
- Тёмные кухни (а таких в каталоге ~30%) теряются на тёмном фоне
- Мобильные устройства на солнце -- тёмный фон плохо читается (77% трафика -- смартфоны)
- Печать страниц -- расход тонера, нечитаемые распечатки

**Риск:** Для среднего ценового сегмента (175K-429K) полностью тёмный сайт может казаться слишком серьёзным и отталкивать «домашнюю» аудиторию (молодые семьи, новосёлы -- основная ЦА). Кухня = уют, а не ночной клуб.

---

### Направление B: «Тёплый светлый» (Warm Light)

**Концепция:** Инвертируем: светлый тёплый фон как база, тёмные и терракотовые акценты.

**Палитра:**
- Основной фон: #FDFAF6 (тёплый белый, как в текущем brand-guide)
- Альтернативный фон: #F5EDE3 (тёплый песочный для чередования)
- Текст: #2D2418 (тёмно-коричневый)
- Primary: #C2613A (терракот)
- Secondary: #3D6B5E (лесной зелёный)
- Карточки: #FFFFFF с border #E0D5C7

**Визуальная логика:**
- Все секции светлые, чередуются: белый / кремовый / песочный
- Фото кухонь в естественном свете -- гармонируют со светлым фоном
- CTA-кнопки терракотовые -- заметны, но не агрессивны
- Типографика тёмно-коричневая, читается легко

**Что сохраняется от MEBELIT:** структура блоков, типы секций, карточки, квиз, команда.

**Что улучшается:** читаемость, ощущение тепла и уюта (прямая ассоциация с домашней кухней), mobile-friendliness.

**Плюсы:**
- Лучше для SEO-контента (блог, описания, FAQ) -- длительное чтение без усталости глаз
- Ассоциация «тёплый светлый = уютная кухня» -- прямая эмоциональная связь
- Лучше на мобильных устройствах при любом освещении (77% трафика)
- Проще обеспечить WCAG AA контраст для всех элементов
- Фото кухонь любого цвета (и светлые, и тёмные) хорошо смотрятся на светлом

**Минусы:**
- Уход от тёмного характера MEBELIT -- существующие клиенты могут не узнать сайт
- Менее драматичный контраст на первом экране
- Визуально может сливаться с конкурентами (Alkor, Rolen -- тоже светлые)
- Потеря ощущения «премиальности», которое даёт тёмный фон

**Риск:** Полная инверсия (тёмный → светлый) может разорвать визуальную связь с mebelit.site. Для аудитории, которая уже знает MEBELIT (5 286 уников за 90 дней), новый сайт будет выглядеть как «совершенно другая компания», а не эволюция.

### Направление C: «Гибридный» (Dark Hero + Light Content)

**Концепция:** Тёмный Hero и акцентные секции (как MEBELIT), светлые контентные блоки (каталог, тексты, блог). Ритмичное чередование.

**Палитра:**
- Hero/акцентные секции: #1C1C1C (тёплый чёрный)
- Контентные секции: #FDFAF6 (тёплый белый) / #F5EDE3 (песочный)
- CTA-блоки и футер: #2D2418 (тёмно-коричневый)
- Текст на тёмном: #F5EDE3 (тёплый белый)
- Текст на светлом: #2D2418 (тёмно-коричневый)
- Primary: #C2613A (терракот)
- Secondary: #3D6B5E (лесной зелёный)
- Карточки: #FFFFFF с тонким border #E0D5C7

**Визуальная логика:**
```
[ТЁМНЫЙ] Hero: фото кухни + overlay + оффер + CTA          ← как MEBELIT
[СВЕТЛЫЙ] Преимущества: 4 блока с иконками на кремовом фоне
[СВЕТЛЫЙ] Каталог: карточки кухонь на белом фоне            ← читаемость
[ТЁМНЫЙ] CTA-полоса: «Рассчитать стоимость» на тёмном      ← акцент
[СВЕТЛЫЙ] Отзывы: реальные фото + текст на светлом
[СВЕТЛЫЙ] Команда: фото + описание на светлом
[ТЁМНЫЙ] Квиз: 4 шага на тёмном фоне                       ← как MEBELIT
[ТЁМНЫЙ] Футер: контакты + навигация на тёмно-коричневом
```

**Что сохраняется от MEBELIT:** тёмный Hero (драматичный первый экран), тёмный квиз, тёмный футер -- три ключевых точки.

**Что улучшается:** контентные блоки (каталог, отзывы, описания) на светлом -- легко читаются. Ритмичное чередование тёмный/светлый создаёт визуальный ритм и разбивает монотонность.

**Плюсы:**
- Лучший баланс: драматичный первый экран (как MEBELIT) + удобство чтения контента
- Тёмный Hero и квиз создают визуальные «рамки» страницы -- узнаваемый почерк
- Каталог на светлом фоне: все кухни (и светлые, и тёмные) хорошо видны
- Блог и описания на светлом -- длительное чтение без усталости
- Переходы тёмный/светлый создают ритм и не дают скроллу быть скучным
- Сохраняет «серьёзность» MEBELIT на ключевых экранах, добавляет «тепло» на контентных

**Минусы:**
- Нужно аккуратно проектировать переходы между тёмным и светлым (без резких стыков)
- Два набора стилей текста (светлый на тёмном, тёмный на светлом) -- больше CSS
- При плохой реализации может выглядеть как «два разных сайта склеенных вместе»

**Риск:** Переходы между тёмными и светлыми секциями требуют дизайнерского мастерства. Решение: использовать мягкие градиентные переходы (не резкую линию), единую сетку и типографику, сквозные акцентные цвета (терракот + зелёный) на обоих фонах.

## 4. Рекомендация

### Выбор: Направление C -- «Гибридный» (Dark Hero + Light Content)

#### Почему C лучше для «Кухни Рема»

**1. Сохраняет ДНК MEBELIT на ключевых экранах.**
Тёмный Hero -- это то, что клиенты запоминают первым. 4 828 визитов из рекламы попадают именно на Hero. Если первый экран будет тёмным с фото кухни и терракотовым CTA -- существующая аудитория (5 286 уников) узнает «свой» сайт. Тёмный квиз и футер усиливают эту связь.

**2. Решает главную проблему направления A (тёмный целиком).**
77% трафика -- смартфоны. Полностью тёмный сайт на мобильном при дневном свете -- плохая читаемость. Гибрид: Hero тёмный (одна-две прокрутки), дальше светлый контент -- комфортное чтение. Каталожные страницы `/straight` и `/corner` (отказы 11-13% -- лучшие на сайте) будут на светлом фоне, сохраняя и улучшая эти показатели.

**3. Решает главную проблему направления B (светлый целиком).**
Полностью светлый сайт не отличается от конкурентов (Alkor, Rolen, Кухни-Красноярск -- все на светлом). Тёмный Hero + квиз делают «Кухни Рема» визуально уникальным в красноярском рынке. Ни один конкурент не использует гибридную схему.

**4. Восприятие цены (средний сегмент 175K-429K).**
Тёмный Hero говорит: «мы серьёзные, не дешёвка». Светлый каталог с ценами говорит: «мы прозрачные, нам нечего скрывать». Именно этот баланс нужен для среднего+ сегмента. Полностью тёмный (A) -- рискует казаться «дорого» для бюджета 175K. Полностью светлый (B) -- рискует казаться «просто ещё одна кухня».

**5. Маркетинг и конверсия.**
- Hero (тёмный, драматичный) -- захват внимания из рекламы. Конверсия в квиз через контрастный CTA
- Каталог (светлый, чистый) -- удержание и выбор. Карточки читаются, цены видны, фото не теряются
- CTA-полоса (тёмная, акцентная) -- повторный призыв к действию между блоками
- Квиз (тёмный) -- фокус на пошаговом заполнении, убирает отвлекающие элементы
- Ритм тёмный/светлый -- не даёт глазу «привыкнуть» и пропустить секцию

**6. SEO и блог.**
Блог, FAQ, описания моделей -- на светлом фоне. Длительное чтение, хорошая читаемость, комфорт для поисковых роботов (хотя роботам всё равно, пользователям нет). Органика сейчас 0.2% (18 визитов / 90 дней) -- нужно растить, а текстовый контент на светлом работает лучше.

#### Почему A и B слабее

| Критерий | A (Чистый тёмный) | B (Тёплый светлый) | **C (Гибридный)** |
|----------|-------------------|--------------------|--------------------|
| Преемственность MEBELIT | Максимальная | Минимальная | **Оптимальная** |
| Мобильная читаемость | Плохая | Отличная | **Хорошая** |
| Уникальность среди конкурентов | Высокая | Низкая | **Высокая** |
| Блог/SEO-контент | Плохо | Отлично | **Отлично** |
| Ассоциация с кухней/уютом | Слабая | Сильная | **Сбалансированная** |
| Восприятие ценового сегмента | Дорого | Нейтрально | **Средний+** |
| Конверсия Hero | Высокая | Средняя | **Высокая** |
| Сложность реализации | Низкая | Низкая | **Средняя** |

Направление A слишком упирается в «тёмный = серьёзный» и игнорирует 77% мобильного трафика. Направление B слишком сильно рвёт с MEBELIT и теряется среди конкурентов. Направление C берёт лучшее от обоих.

#### Переходы между тёмным и светлым

Для избежания эффекта «два склеенных сайта»:

- **Приём 1: Градиентный переход.** Нижний край тёмной секции -- gradient от `#1C1C1C` к `#FDFAF6` высотой 80-120px
- **Приём 2: Диагональный clip-path.** Нижний край тёмной секции обрезается по диагонали (2-3 градуса), светлая секция «заходит» под неё
- **Приём 3: Декоративный разделитель.** Тонкая полоса терракотового (#C2613A) высотой 4px между секциями -- сквозной элемент, связывающий тёмное и светлое

**Рекомендация:** Приём 3 (терракотовая полоса) -- самый простой в реализации, стабильно работает на мобильных, создаёт визуальную связь через единый акцентный цвет.

### Контрастность (WCAG AA)

| Комбинация | Контраст | WCAG AA |
|-----------|----------|---------|
| `#F5EDE3` на `#1C1C1C` | 13.8:1 | Проходит (норма 4.5:1) |
| `#C2613A` на `#1C1C1C` | 3.7:1 | Проходит для крупного текста (норма 3:1) |
| `#C2613A` на `#FDFAF6` | 3.8:1 | Проходит для крупного текста; для мелкого использовать `--color-primary-dark` #9E4A2A (5.3:1) |

| `#FFFFFF` на `#9E4A2A` | 5.4:1 | Проходит для любого текста |

**Вывод:** Для мелкого текста ссылок на светлом фоне использовать `--color-primary-dark` (#9E4A2A) вместо `--color-primary` (#C2613A). Для CTA-кнопок (крупный текст) -- `--color-primary` допустим.
