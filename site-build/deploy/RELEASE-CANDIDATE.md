## 2. Состав RC

### 2.1. WordPress-тема `kuhni-rema`

- **37 файлов**, ~7 800 строк кода
- Дочерняя тема для Bricks Builder

### 2.2. Пользовательские типы записей (CPT) и ACF-поля

| CPT | Назначение | ACF-поля |
|-----|-----------|----------|
| Kitchen | Каталог кухонь | Цена, размеры, материалы, фото галерея, стиль |
| Project | Портфолио выполненных проектов | Фото до/после, описание работ, сроки |
| Review | Отзывы клиентов | Имя, текст, оценка, фото, источник |
| Team | Команда | Должность, фото, опыт |
| Promotion | Акции и спецпредложения | Сроки действия, условия, баннер |
| FAQ | Часто задаваемые вопросы | Вопрос, ответ, категория |

### 2.3. Шаблоны страниц

12 шаблонов страниц для ключевых разделов сайта (главная, каталог, карточка кухни, портфолио, отзывы, акции, о компании, контакты, FAQ, квиз, калькулятор, политика конфиденциальности).

### 2.4. CSS дизайн-система

| Файл/слой | Назначение |
|-----------|-----------|
| Tokens | Цвета, шрифты, отступы, тени — дизайн-токены |
| Base | Сброс стилей, типографика, базовые элементы |
| Components | Стили компонентов (карточки, кнопки, формы, навигация) |
| Responsive | Адаптив для мобильных, планшетов, десктопа |

### 2.5. JavaScript

| Модуль | Назначение |
|--------|-----------|
| `main.js` | Общая логика сайта |
| `forms.js` | Валидация и отправка форм |
| `quiz.js` | Логика квиза подбора кухни |
| `quiz-popup.js` | Попап-версия квиза |
| `analytics.js` | Отправка событий аналитики |
| `datalayer.js` | Data Layer для Яндекс Метрики |

### 2.6. SEO

- **Schema.org** — структурированные данные (LocalBusiness, Product, Review, FAQ, BreadcrumbList)
- **Open Graph мета-теги** — для соцсетей (VK, Telegram)
- **RankMath Pro** — конфигурация SEO-плагина (sitemap, redirects, schema)

### 2.7. Деплой-скрипты

| Скрипт | Назначение |
|--------|-----------|
| `install-wordpress.sh` | Установка WordPress, плагинов, темы на VPS |

| `smoke-test.sh` | Проверка работоспособности после деплоя |
| `nginx-kuhnirema.conf` | Конфигурация Nginx для домена |

### 2.8. GitHub Actions

| Workflow | Назначение |
|----------|-----------|
| `deploy-theme.yml` | Деплой темы на VPS через SSH |

### 2.9. Импорт контента

| Контент | Количество |
|---------|-----------|

## 4. Известные ограничения и риски

### 4.1. Требуют ручной настройки при деплое

| Ограничение | Описание | Решение |
|------------|----------|---------|
| Bricks Builder шаблоны | Шаблоны требуют ручной настройки в визуальном редакторе Bricks — программный импорт макетов невозможен | Настроить вручную после установки темы, используя документацию шаблонов |
| ACF Pro лицензия | Плагин ACF Pro требует лицензионный ключ для обновлений и поддержки | Ввести ключ в настройках плагина после установки |
| RankMath Pro лицензия | Плагин RankMath Pro требует лицензионный ключ | Ввести ключ в настройках плагина после установки |
| SMTP | Почтовый сервер не настроен — формы не будут отправлять email | Настроить WP Mail SMTP или аналог при деплое |

### 4.2. Внешние зависимости

| Риск | Описание | Митигация |
|------|----------|----------|
| Фото из VK | При импорте кухонь фотографии скачиваются через VK API — зависимость от доступности VK | Предварительно скачать и сохранить медиа локально перед деплоем |
| Google APIs из Claude Code | IP-адреса среды Claude Code получают 403 от Google APIs | Все операции с Google Sheets выполнять через GitHub Actions (`sheets-manage.yml`) |

### 4.3. Инфраструктурные

| Риск | Описание | Рекомендация |
|------|----------|-------------|
| Яндекс Метрика | Счётчики показывают `CS_ERR_UNKNOWN` — требуется диагностика | Проверить корректность ID счётчика и код вставки после деплоя |
| Нет CDN | Статика отдаётся напрямую с VPS, нет кеширующего CDN | Рассмотреть подключение Cloudflare или аналога после запуска |
| Нет бэкап-стратегии | Резервное копирование БД и файлов не настроено | Настроить автобэкапы (UpdraftPlus или серверный cron) при деплое |

## 5. Критерии приёмки RC

RC-1.0.0 считается **принятым**, если выполнены все следующие условия:

### 5.1. Деплой

- [ ] WordPress успешно установлен на VPS
- [ ] Тема `kuhni-rema` активирована
- [ ] Плагины ACF Pro, RankMath Pro, Bricks Builder установлены и активированы
- [ ] Лицензионные ключи введены
- [ ] Nginx настроен, SSL-сертификат выпущен (Let's Encrypt)
- [ ] Домен кухнирема.рф указывает на VPS

### 5.2. Контент

- [ ] Все 42 кухни импортированы с фото и ACF-полями
- [ ] Все 12 проектов импортированы
- [ ] Все 15 отзывов импортированы
- [ ] Все 18 FAQ импортированы
- [ ] Шаблоны Bricks Builder настроены для всех 12 страниц

### 5.3. Функциональность

- [ ] Главная страница загружается за < 3 сек
- [ ] Каталог кухонь отображает все записи с фильтрацией
- [ ] Карточка кухни показывает все данные (цена, фото, описание)
- [ ] Формы отправляют заявки через n8n webhook
- [ ] Квиз подбора кухни работает от начала до конца
- [ ] Адаптивная вёрстка корректна на мобильных (320px+), планшетах и десктопе
- [ ] Телефонные ссылки кликабельны на мобильных

### 5.4. SEO

- [ ] Schema.org разметка валидна (проверка через Google Rich Results Test)
- [ ] Open Graph теги корректны (проверка через VK/Telegram превью)
- [ ] RankMath показывает зелёный балл для ключевых страниц
- [ ] XML Sitemap доступен и содержит все страницы
- [ ] 301-редиректы с mebelit.site настроены и работают
- [ ] robots.txt корректен

### 5.5. Аналитика

- [ ] Яндекс Метрика счётчик установлен и передаёт данные
- [ ] Цели Метрики срабатывают (проверить минимум 3 из 12)
- [ ] Data Layer корректно передаёт события

### 5.6. Smoke-тест

- [ ] `smoke-test.sh` проходит без ошибок
- [ ] Все ключевые URL возвращают HTTP 200
- [ ] Нет ошибок PHP в логах
- [ ] Нет JS-ошибок в консоли браузера

### 5.7. Безопасность

- [ ] WordPress обновлён до актуальной версии
- [ ] Стандартный путь wp-login.php защищён или изменён
- [ ] SMTP настроен (формы отправляют email)
- [ ] Бэкап-стратегия настроена и протестирована
