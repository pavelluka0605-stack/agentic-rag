## 2. Лидогенерация

### Текущие проблемы

- ~10 CTA-точек на mebelit.site — перегруз, создают давление вместо вовлечения
- Глубина просмотра 1.37 страницы — пользователи не вовлекаются
- Конверсия в заявку неизвестна (нет данных)
- Квиз получает только 4.7% трафика (343 из 7 287), хотя это лучший элемент (19.5% отказов)

### 2.1 Обязательные формы (4 штуки)

| # | Форма | Где размещена | Поля | Webhook Action | Приоритет |
|---|-------|---------------|------|----------------|-----------|
| 1 | **Квиз-калькулятор** | Главная (встроенный), /kalkulyator/ (полная страница), попап по времени | 5 шагов (см. 2.3) | `POST /webhook/quiz-lead` → n8n → Google Sheets «Заказы_клиентов» + Telegram менеджеру | Критический |
| 2 | **Запись на замер** | /kontakty/, /besplatnyj-zamer/, footer всех страниц | Имя, телефон, адрес (опционально), удобное время | `POST /webhook/zamer-lead` → n8n → Google Sheets + Telegram | Критический |
| 3 | **Загрузить проект конкурента** | /pryamye-kuhni/, /uglovye-kuhni/, /p-obraznye-kuhni/, детальные карточки | Имя, телефон, файл (jpg/pdf, до 10MB) | `POST /webhook/competitor-project` → n8n → Google Sheets + Telegram + файл в хранилище | Высокий |
| 4 | **Обратный звонок** | Хедер (все страницы), sticky кнопка мобильная | Телефон, имя (опционально) | `POST /webhook/callback` → n8n → Google Sheets + Telegram | Высокий |

Все формы: AJAX-отправка без перезагрузки страницы, редирект на /spasibo/ после успешной отправки. Валидация: телефон (маска +7), обязательные поля.

### 2.4 Баланс форм

| Правило | Значение | Обоснование |
|---------|----------|-------------|
| Макс. форм на страницу | 2 (1 primary + 1 secondary) | Было ~10 CTA на mebelit.site — перегруз. Исследования: >3 CTA снижают конверсию |
| Sticky CTA (мобильная) | Да, внизу экрана | Кнопка «Рассчитать» + иконка телефона. Появляется после скролла 300px. 76.7% трафика — мобильный |
| Exit-intent попап | Нет | Удалить. Агрессивные попапы раздражают, увеличивают отказы |
| Попап по времени | Да, 1 раз за сессию, через 45 сек | Предложение квиза: «Рассчитайте стоимость кухни за 5 минут». Показывать только если не взаимодействовал с формой |
| Попап по скроллу | Нет | Заменяется инлайн-CTA в контенте |
| Cookie: не показывать повторно | 7 дней | Если закрыл попап или уже заполнил форму — не показывать 7 дней |
| Телефон в хедере | Да, кликабельный | +7 (391) 216-97-59, отслеживается как цель Метрики |


|---------------|-------------|-------------|

#### Правила UTM

1. Все значения — латиница, нижний регистр, слова через дефис или подчёркивание
2. Без кириллицы в UTM (проблемы с кодировкой в Метрике и n8n)
3. Каждая новая кампания получает уникальный utm_campaign
4. utm_content — для A/B тестирования креативов
5. utm_term — для ключевых слов (Директ) или описания таргетинга (VK)
