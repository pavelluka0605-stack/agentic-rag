### 1.3 Динамические элементы

| Элемент | Где генерируется | Технология | Зачем |

### 1.4 Управляемость (что редактируется из админки БЕЗ разработчика)

| Действие | Где в админке | Как |
|----------|--------------|-----|
| Добавить новую кухню | Кухни > Добавить кухню | Заполнить 14 ACF-полей (цена, размеры, галерея, фурнитура и т.д.) + выбрать таксономии (тип, стиль, материал) |
| Изменить цену кухни | Кухни > Редактировать > поле "Цена" | Число в рублях. Цена со скидкой -- отдельное поле. Перечёркивание старой цены автоматически |
| Добавить/удалить фото | Кухни > Редактировать > Галерея фото | Drag-and-drop загрузка. Мин. 800x600px. WebP/JPEG |
| Изменить тексты на страницах | Страницы > Редактировать | Bricks Builder visual editor |
| Создать/закрыть акцию | Акции > Добавить/Редактировать | Заголовок, описание, даты, баннер, переключатель "Активна". Автоматическое снятие по дате |
| Добавить отзыв | Отзывы > Добавить | Имя, текст, оценка 1-5, фото, источник (Flamp/2ГИС/сайт). Schema.org Review автоматически |
| Добавить FAQ | FAQ > Добавить | Вопрос, ответ, категория. Schema.org FAQPage автоматически |
| Добавить проект в портфолио | Портфолио > Добавить | Название, описание, галерея до/после, тип кухни, площадь, срок, отзыв клиента |
| Изменить контакты | Настройки > Контакты (ACF Options Page) | Телефон, адрес, email, часы работы, координаты -- меняются глобально на всём сайте |
| Управление квизом | Настройки > Квиз (ACF Options Page) | Шаги, варианты ответов, тексты кнопок |
| Управление командой | Команда > Добавить/Редактировать | Имя, должность, фото, описание, порядок сортировки |

**Важно:** Все изменения контента доступны через ACF-интерфейс в WordPress-админке. Для изменения структуры страниц (блоки, секции) нужен Bricks Builder. Для правок через чат -- chat.mebelit.ru (Claude Chat PWA).

| **Скриншоты отзывов из 2ГИС/Flamp** | `/otzyvy/` (блок "Отзывы на площадках") | Внешние площадки = невозможно подделать. Даже смешанные отзывы (хвалят качество, ругают сроки) работают на честность |

На кухнирема.рф -- центральное позиционирование |

**Принцип размещения дожимающих блоков:**
- **Hero (top):** УТП "1 см" + главный CTA -- первое впечатление
- **Mid:** рассрочка + акция + "пришлите проект" -- после ознакомления с каталогом, когда цена уже видна
- **Bottom:** "бесплатный замер" + "установка за 1 день" -- перед уходом со страницы
- **Sticky (mobile):** "Рассчитать стоимость" -- всегда доступен, не мешает просмотру
