# Setup Guide — mebelit.site WordPress v2.0

Пошаговая инструкция по установке и настройке WordPress-сайта mebelit.site.

---

## 1. Установка WordPress

1. Развернуть WordPress 6.x на хостинге
2. Настроить домен `mebelit.site` → DNS A-записи
3. Установить SSL-сертификат (Let's Encrypt)
4. Настроить `wp-config.php`:
   - DB credentials
   - `define('WP_MEMORY_LIMIT', '256M');`
   - `define('DISALLOW_FILE_EDIT', true);`

## 2. Установка плагинов

### Обязательные:
| Плагин | Версия | Назначение |
|--------|--------|------------|
| Elementor Pro | Latest | Page builder |
| ACF Pro | Latest | Custom fields |
| Yoast SEO | Latest | SEO |
| WP Mail SMTP | Latest | Доставка email |

### Рекомендуемые:
| Плагин | Назначение |
|--------|------------|
| LiteSpeed Cache / WP Super Cache | Кеширование |
| UpdraftPlus | Бэкапы |
| Redirection | Редиректы со старых URL |

### НЕ устанавливать:
- Contact Form 7 / WPForms (формы реализованы через тему)
- Wordfence (security headers уже в теме)
- Любые "booster" / "optimizer" плагины без необходимости

## 3. Установка темы

1. Установить родительскую тему **Hello Elementor**
2. Загрузить дочернюю тему:
   ```
   Внешний вид → Темы → Загрузить → ZIP архив mebelit-child/
   ```
3. Активировать **Mebelit Child**

### Создание ZIP:
```bash
cd wordpress-mebelit/theme/
zip -r mebelit-child.zip mebelit-child/
```

## 4. Импорт ACF полей

1. Перейти: ACF → Инструменты → Импорт
2. Загрузить файл: `acf/acf-field-groups.json`
3. Проверить: 4 группы полей (Кухни, Отзывы, Настройки, Команда)

## 6. Создание страниц

| Страница | Slug | Шаблон |
|----------|------|--------|
| Главная | (Front Page) | Elementor Full Width |
| Кухни прямые | straight | Elementor Full Width |
| Кухни угловые | corner | Elementor Full Width |
| Кухни П-образные | pshape | Elementor Full Width |
| Квиз | quiz | Квиз — Калькулятор |
| Политика конфиденциальности | politics | Политика конфиденциальности |

### Настройка:
- Настройки → Чтение → Статическая страница → Главная: "Главная"
- Настройки → Постоянные ссылки → "Название записи" (`/%postname%/`)

## 7. Создание меню

- Внешний вид → Меню → "Главное меню":
  - Кухни прямые → /straight/
  - Кухни угловые → /corner/
  - Кухни П-образные → /pshape/

## 8. Импорт контента кухонь

Используя данные из `content/kitchens-data.json`, создать записи CPT Kitchen.

**Прямые кухни** (таксономия: straight): 8 записей
**Угловые кухни** (таксономия: corner): 9 записей
**П-образные кухни** (таксономия: pshape): 8 записей

Характеристики по умолчанию для всех:
- Сроки: от 21 дня
- Гарантия: 24 месяца
- Бесплатный выезд дизайнера

Для каждой кухни загрузить фото (Featured Image).

## 9. Импорт отзывов

Используя `content/reviews-data.json`, создать 10 записей CPT Review.

## 10. Сборка в Elementor

Следовать инструкции: `elementor-templates/ELEMENTOR-BUILD-GUIDE.md`

### Порядок:
1. Header (Theme Builder)
2. Footer (Theme Builder)
3. Главная страница
4. Страницы каталога (straight, corner, pshape)
5. Проверка квиза (/quiz/) и политики (/politics/)

## 13. DNS и редиректы

1. DNS A-запись для mebelit.site → IP сервера
2. www → mebelit.site (CNAME или редирект)
3. Редиректы:
   - `/pryamye-kuhni/` → `/straight/`
   - `/uglovye-kuhni/` → `/corner/`
   - `/p-obraznye-kuhni/` → `/pshape/`
   - `/politika-konfidencialnosti/` → `/politics/`

## 14. Webhook интеграция

1. В n8n создать Webhook Trigger
2. В WordPress: Mebelit → Настройки → Webhook URL → вставить URL
3. Проверить: отправить тестовую заявку
