# Руководство по установке WordPress — mebelit.site

## Шаг 1: Установка WordPress

### На хостинге (рекомендуется)
1. Установи WordPress 6.x на хостинг mebelit.site
2. Выбери русский язык при установке
3. Создай админа (НЕ admin, используй своё имя)

### На VPS
```bash
# Если используешь VPS, установи через Docker
docker run -d \
  --name wordpress \
  -p 80:80 \
  -e WORDPRESS_DB_HOST=db \
  -e WORDPRESS_DB_NAME=mebelit \
  -e WORDPRESS_DB_USER=mebelit \
  -e WORDPRESS_DB_PASSWORD=<strong_password> \
  wordpress:latest
```

## Шаг 2: Установка плагинов

### Обязательные (6 шт.)
1. **Elementor Pro** — конструктор страниц (лицензия)
2. **Advanced Custom Fields PRO** — пользовательские поля (лицензия)
3. **Yoast SEO** — SEO (бесплатная версия достаточна)
4. **WP Mail SMTP** — надёжная отправка email
5. **LiteSpeed Cache** или **WP Super Cache** — кеширование
6. **Wordfence** — базовая безопасность (бесплатная версия)

### Опциональные
- **WPForms Lite** — если не хватает Elementor Forms
- **Imagify** или **ShortPixel** — сжатие изображений
- **UpdraftPlus** — бэкапы

### НЕ ставить
- Всё остальное. Минимум плагинов = максимум скорости.

## Шаг 3: Установка темы

1. Зайди в **Внешний вид → Темы → Добавить**
2. Установи и активируй **Hello Elementor**
3. Загрузи папку `mebelit-child/` из `wordpress-mebelit/theme/` как ZIP
4. Активируй **Mebelit Child**

### Как создать ZIP из папки:
```bash
cd wordpress-mebelit/theme/
zip -r mebelit-child.zip mebelit-child/
```

## Шаг 4: Настройка ACF

1. Зайди в **ACF → Инструменты → Импорт**
2. Загрузи файл `acf/acf-field-groups.json`
3. Импортируй все группы полей

## Шаг 5: Создание страниц

### Страницы WordPress:
| Страница | Slug | Шаблон |
|----------|------|--------|
| Главная | / | Elementor Full Width |
| Прямые кухни | /straight/ | Elementor Full Width |
| Угловые кухни | /corner/ | Elementor Full Width |
| П-образные кухни | /pshape/ | Elementor Full Width |
| Квиз | /quiz/ | Квиз — Калькулятор стоимости |
| Политика | /politics/ | Default |

### Настройка главной:
1. **Настройки → Чтение** → Статическая страница → Главная

### Настройка постоянных ссылок:
1. **Настройки → Постоянные ссылки** → Название записи (`/%postname%/`)

## Шаг 6: Создание меню

1. **Внешний вид → Меню**
2. Создай меню «Главное меню»
3. Добавь страницы: Прямые кухни, Угловые кухни, П-образные кухни
4. Назначь для **Header Menu**

## Шаг 7: Импорт контента

### Кухни (CPT: kitchen)
1. Зайди в **Кухни → Добавить**
2. Для каждой кухни из `content/kitchens-data.json`:
   - Заголовок = название кухни
   - Загрузи фото (скачай с Tilda CDN)
   - Заполни ACF поля: цена, характеристики
   - Выбери тип: прямая / угловая / п-образная

### Отзывы (CPT: review)
1. Зайди в **Отзывы → Добавить**
2. Для каждого отзыва:
   - Заголовок = имя автора
   - ACF поля: автор, текст, рейтинг (5)

### Команда
1. Зайди в **Mebelit → Команда**
2. Добавь 3 участников:
   - Наоми — дизайнер
   - Павел — инженер-дизайнер
   - Николай Колесников — инженер-дизайнер

### Настройки сайта
1. Зайди в **Mebelit → Настройки сайта**
2. Заполни: телефон, адрес, часы работы, webhook URL

## Шаг 8: Сборка в Elementor

### Header (Elementor Theme Builder)
1. **Шаблоны → Theme Builder → Header**
2. Создай новый header:
   - Логотип (слева)
   - Меню навигации (центр)
   - Телефон + CTA кнопка (справа)
3. Фон: `var(--color-primary)` (#1a1a2e)
4. Sticky header

### Footer (Elementor Theme Builder)
1. **Шаблоны → Theme Builder → Footer**
2. 3 колонки:
   - Логотип + копирайт
   - Контакты (телефон, адрес, часы)
   - CTA кнопка + ссылка на политику

### Главная страница
Собери в Elementor по структуре из `content/pages-content.md`:
1. Hero Section (Full Width, тёмный фон, фото справа)
2. Акция (рассрочка + скидка)
3. Каталог категорий (3 карточки)
4. Преимущества (4 иконки + текст)
5. Форма заявки
6. Дизайнер (3 пункта + CTA)
7. Гарантии (3 карточки)
8. До/После (Image Comparison)
9. Калькулятор CTA
10. Команда (3 карточки)
11. Отзывы (Carousel)
12. Квиз CTA

### Страницы каталога
Используй одну структуру для всех 3:
1. Hero
2. Гарантия цены
3. Карточки кухонь (Posts Widget → CPT kitchen, фильтр по taxonomy)
4. Удобства
5. Форма заявки

## Шаг 9: Настройка форм

### Elementor Forms (или WPForms)
1. Создай 3 формы:
   - **Быстрая заявка** (имя + телефон)
   - **Вызов дизайнера** (имя + телефон)
   - **Обратный звонок** (телефон)

2. Actions After Submit:
   - Email → admin
   - Webhook → `{webhook_url}` (JSON)

### Webhook формат:
```json
{
  "name": "Имя",
  "phone": "+7...",
  "form_type": "quick_request",
  "page_url": "https://mebelit.site/",
  "timestamp": "2025-01-01 12:00:00"
}
```

## Шаг 11: Скорость и безопасность

1. **LiteSpeed Cache** — включить кеширование страниц
2. **Imagify** — сжать все загруженные изображения
3. **Wordfence** — включить Firewall, настроить 2FA для админа
4. **SSL** — убедиться что HTTPS работает
5. **Бэкап** — настроить UpdraftPlus (ежедневно → Google Drive / Dropbox)

## Шаг 12: Редиректы и DNS

1. Убедись что старые URL перенаправляют на новые:
   - /straight → /straight/ (trailing slash)
   - /corner → /corner/
   - /pshape → /pshape/
   - /politics → /politics/
   - /quiz → /quiz/

2. DNS: направь mebelit.site → IP хостинга WordPress
